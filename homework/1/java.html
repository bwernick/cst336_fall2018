What makes it important?

Java is used to develop enterprise-level applications. It also allows you to develop web-based applications. When used online, Java allows applets to be used and downloaded through a browser.

Programs that use or are written in Java include the Adobe Creative Suite, Eclipse IDE, Minecraft, and OpenOffice.

Some features of the language are:

  • Application portablility
  • Robust and interpreted language
  • Extensive Network Library
